## User Schema

| Key         |    Type     |                       Description |
| :---------- | :---------: | --------------------------------: |
| id          |   String    |                                id |
| name        |   String    |                Name of the Person |
| email       |   String    |               email of the person |
| image       |   String    |                    avatar img url |
| nationality |   String    |            the tld of the country |
| age_range   |     Int     | [Age range options](#age-options) |
| gender      |     Int     | [Gender options](#gender-options) |
| createdAt   |   String    |                     creation time |
| updatedAt   |   String    |                       update time |
| Booking     | Booking [ ] |                     booking array |

### <a id="age-schema"></a>age range options

| index | Description |
| :---: | ----------: |
|   0   |      0 - 17 |
|   1   |     18 - 24 |
|   2   |     25 - 30 |
|   3   |     31 - 40 |
|   4   |         41+ |

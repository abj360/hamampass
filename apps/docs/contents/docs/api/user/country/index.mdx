## Country Schema

| Key     |  Type  |                     Description |
| :------ | :----: | ------------------------------: |
| tld     | String | id also the tld for the country |
| name_tr | String |     Turkish Name of the Country |
| name_en | String |     English Name of the Country |
| image   | String |                img url for flag |

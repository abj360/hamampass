## Amentiy Schema

| Key          |     Type     |                                     Description |
| :----------- | :----------: | ----------------------------------------------: |
| id           |    String    |                                              id |
| items        |   Int [ ]    |             [Item Array Options](#Item-options) |
| facilities   |   Int [ ]    | [Facilities Array Options](#Facilities-options) |
| foods_drinks |   Int [ ]    |           [Foods Array Options](#Foods-options) |
| createdAt    |    String    |                                   creation time |
| updatedAt    |    String    |                                     update time |
| Property     | Property [ ] |                                  property array |

## Contact Schema

| Key       |     Type     |                   Description |
| :-------- | :----------: | ----------------------------: |
| id        |    Strig     |                            id |
| city      |    String    |    Name of the city in turkey |
| district  |    String    |  Name of the district in city |
| phone     |    String    |  phone number including "+90" |
| map_link  |    String    | google maps link of the place |
| address   |    String    |      Full adress of the place |
| location  |  Float [ ]   |  [ lat: float , long: float ] |
| createdAt |    String    |                 creation time |
| updatedAt |    String    |                   update time |
| Property  | Property [ ] |                property array |

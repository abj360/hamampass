## Property Schema

| Key          |    Type     |                   Description |
| :----------- | :---------: | ----------------------------: |
| id           |     Int     |                            id |
| title        |   String    |             Name of the Hamam |
| sex          |     Int     | 0: women , 1: men , 2: unisex |
| outsider_sex |  Json [ ]   |           sex: Int , day: Int |
| photos       | String [ ]  |               The data itself |
| pay          |     Int     |             0: cash , 1: card |
| desc_tr      |   String    |           Turkish description |
| desc_en      |   String    |           English description |
| contactId    |   String    |            the id for contact |
| hourId       |   String    |               the id for hour |
| ratingId     |   String    |             the id for rating |
| amenityId    |   String    |          the id for amenityId |
| createdAt    |   String    |                 creation time |
| updatedAt    |   String    |                   update time |
| products     | Product [ ] |                 product array |

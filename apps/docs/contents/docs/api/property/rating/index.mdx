## Rating Schema

| Key                  |     Type     |                              Description |
| :------------------- | :----------: | ---------------------------------------: |
| id                   |    String    |                                       id |
| count                |     Int      |                 total numbers of reviews |
| rate_overall         |    Float     | overall rate for all parameters out of 5 |
| rate_location        |    Float     |           overall location rate out of 5 |
| rate_staff           |    Float     |              overall staff rate out of 5 |
| rate_atmosphere      |    Float     |         overall atmosphere rate out of 5 |
| rate_cleanliness     |    Float     |        overall cleanliness rate out of 5 |
| rate_facilities      |    Float     |         overall facilities rate out of 5 |
| rate_value_for_money |    Float     |    overall value_for_money rate out of 5 |
| createdAt            |    String    |                            creation time |
| updatedAt            |    String    |                              update time |
| Property             | Property [ ] |                           property array |

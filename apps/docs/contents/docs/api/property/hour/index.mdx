## Hour Schema

| Key                |     Type     |                           Description |
| :----------------- | :----------: | ------------------------------------: |
| id                 |    String    |                                    id |
| weekdays           |  String [ ]  |         [ "start time" , "end time" ] |
| weekends           |  String [ ]  |         [ "start time" , "end time" ] |
| segregated_details |    Json?     | look up for segregated_details schema |
| createdAt          |    String    |                         creation time |
| updatedAt          |    String    |                           update time |
| Property           | Property [ ] |                        property array |

## Booking Schema

| Key        |   Type   |                       Description |
| :--------- | :------: | --------------------------------: |
| id         |  String  |                                id |
| date       | DateTime |                              date |
| propertyId |  String  |                id of the property |
| userId     |  String  |                    id of the user |
| products   |   Json   | look below schema for description |
| totalMoney |  Float   |      total money for all products |
| review     | Review?  |       the review for that booking |
| createdAt  |  String  |                     creation time |
| updatedAt  |  String  |                       update time |

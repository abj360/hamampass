Base url: https://www.hamampass.com/api

The base url contains information about all available API's resources. All requests are GET or POST requests and go over https. All responses will return data in json.

```jsx {7} showLineNumbers
GET https://www.hamampass.com/api

{
  "property":["https://www.hamampass.com/api/property",
               "https://www.hamampass.com/api/property/{title}"]
}
```

```jsx {7} showLineNumbers
POST https://www.hamampass.com/api

{
  "property":"https://www.hamampass.com/api/property"
}
```

## Info and Pagination

Get all properties and get all reviews API will automatically paginate the responses. You will receive up to 5 documents per page.

Each resource contains an info object with information about the response.

| Key       | Type  |                    Description |
| :-------- | :---: | -----------------------------: |
| all_items |  Int  | Total item count from database |
| page      |  Int  |                   Current Page |
| max_page  |  Int  |           Total count of Pages |
| limit     |  Int  |            item count per page |
| data      | Array |                The data itself |

```jsx {7} showLineNumbers
{
    "all_items": 2,
    "page": 1,
    "max_page": 1,
    "limit": 5,
    "data": [
      ...
    ]
}
```
